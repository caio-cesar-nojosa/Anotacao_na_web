Deixar o meu shell script portável

Maio de 2020

Na primeira linha de uma script normalmente nós colocamos o caminho absoluto do interpretador de comandos, que no meu caso é o bash acopanhado do "#!", como no exemplo abaixo:

#! /bin/bash

Essa não é uma pratica recomendada, pois nem sempre as outras distribuições vão usar esse caminho. A solução é usar o "env".

#! /usr/bin/env bash
